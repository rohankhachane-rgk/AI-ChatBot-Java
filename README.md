# AI ChatBot

## Description

AI ChatBot is a console-based chatbot application developed in Java. The chatbot can respond to user queries, recognize greetings and farewells, answer predefined questions, and learn simple new information during conversations.

## Features

* Interactive chat interface
* Greeting and farewell detection
* Predefined knowledge base
* Learns simple information from users
* Console-based implementation

## Technologies Used

* Java
* Collections Framework
* Object-Oriented Programming (OOP)

## How to Run

1. Compile the program:
   javac AIChatBot.java

2. Run the program:
   java AIChatBot

## Future Improvements

* GUI interface using JavaFX or Swing
* Database integration for persistent learning
* Natural Language Processing (NLP)
* Voice-based interaction
